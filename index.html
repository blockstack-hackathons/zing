Zing

Send money to Twitter and GitHub profiles

View on GitHub Download .zip Download .tar.gz

Installation

  1. Download the zip file then unzip it
  2. Go to chrome://extensions/
  3. Click "load unpacked extension" and select the unzipped folder

Valid Profiles

You can pay any Twitter or GitHub account that has been connected to a blockchain ID. If there's someone you'd like to pay but there is no payment information available for that user, make sure to tell them to sign up for a blockchain ID (e.g. on onename.com) and then verify their GitHub and Twitter accounts.
